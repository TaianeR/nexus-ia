# Nexus IA 🤖

**Nexus IA** é um assistente de inteligência artificial avançado, desenvolvido para responder qualquer tipo de pergunta de forma clara, amigável e em português do Brasil.

## ✦ Funcionalidades

- 💬 Chat interativo com memória de conversa
- 🧠 Inteligência artificial com Google Gemini
- 🌙 Interface premium dark mode
- 📱 Funciona no celular e computador
- ⚡ Respostas rápidas para qualquer assunto

## 🚀 Como usar

Acesse o link do app e comece a conversar!

---

Desenvolvido com ❤️ usando [Streamlit](https://streamlit.io) e [Google Gemini](https://deepmind.google/technologies/gemini/).
